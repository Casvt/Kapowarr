Mega

Mega is a cloud storage service from which Kapowarr can download files. These files are uploaded by GetComics, as an alternative way of downloading the file compared to MediaFire, torrents or the direct downloads from their own servers. An advantage of Mega is that their downloads are end-to-end encrypted, which Kapowarr makes use of. There are free and paid accounts that offer higher bandwidth/limits for downloading files. Kapowarr is able to download files using an account to take advantage of these higher limits (e.g. being able to download more than 5GB per day). If you have a Mega account, it's advised to enter the credentials below so that Kapowarr can take advantage of it's premium features. Also make sure to set Mega as the highest service in the service preference, so that Kapowarr will prefer using it.

Note: Kapowarr can download files from Mega, but not Mega folders (yet).

MediaFire

MediaFire is a cloud storage service from which Kapowarr can download files. These files are uploaded by GetComics, as an alternative way of downloading the file compared to Mega, torrents or the direct downloads from their own servers.

Note: Kapowarr does not support downloading content from MediaFire using an account, like it can with Mega.

Note: Kapowarr can download files from MediaFire, but not MediaFire folders (yet).

GetComics

GetComics is a website that offers a vast amount of downloads for comics. Kapowarr can search this website to find downloads for the volumes in the library. The comics can be downloaded via multiple services, for instance: MediaFire, Mega, torrents (magnet links) and direct downloads from their servers. Kapowarr supports downloading via all the services mentioned before.

Note: When downloading from the servers of GetComics (instead of using an external service), the download speed will be significantly reduced 400MB into the download. This behaviour is set up by the owner of the website, is intentional, and is not planned to be removed.
